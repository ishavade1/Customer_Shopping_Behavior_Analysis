📊 Customer Shopping Behavior Analysis

📌 Overview

This project focuses on analyzing customer shopping behavior to uncover meaningful insights that can support business decision-making. The analysis covers the complete data workflow — from data loading and cleaning to SQL-based analysis, dashboard creation, and reporting.

The goal of this project is to understand customer patterns, product performance, and revenue drivers using real-world data analysis techniques.

📂 Dataset
The dataset contains customer transaction data including:

Customer demographics (Age, Gender, Location)
Purchase details (Item Purchased, Category, Amount)
Shopping behavior (Discounts, Subscription, Frequency)
Product feedback (Review Rating)
Shipping and order-related information

The dataset was cleaned and prepared before analysis to ensure accuracy and consistency.

🛠️ Tools & Technologies Used

Python (Pandas, NumPy) → Data cleaning & EDA
SQL (PostgreSQL) → Business problem solving
Power BI → Interactive dashboard
Gamma → Presentation (PPT creation)
Jupyter Notebook → Analysis workflow

🔄 Project Workflow / Steps

1. Data Loading
Loaded dataset using Python (Pandas)
Explored structure using .info() and .describe()

2. Data Cleaning
Handled missing values
Renamed columns for consistency
Removed unnecessary columns
Ensured proper data types

3. Exploratory Data Analysis (EDA)
Analyzed customer demographics
Identified purchasing patterns
Explored relationships between variables
Visualized trends using charts

4. SQL Analysis (PostgreSQL)
Imported cleaned data into PostgreSQL
Solved multiple business questions using:
Aggregations
Subqueries
CASE statements
Window functions

5. Dashboard Creation (Power BI)
Built interactive dashboard to visualize:
Revenue trends
Customer segmentation
Product performance
Discount impact

6. Reporting & Presentation
Created a detailed project report
Designed a presentation using Gamma for storytelling

📊 Dashboard Highlights

The Power BI dashboard includes:

Total Revenue and Average Spend
Revenue by Gender and Age Group
Top Performing Products
Customer Segmentation (New, Returning, Loyal)
Impact of Discounts on Sales

👉 The dashboard provides a clear and interactive view of key business metrics.

📈 Key Results & Insights

Identified top revenue-generating customer segments
Found high-performing products based on ratings and sales
Observed the impact of discounts on purchasing behavior
Analyzed differences between subscribed and non-subscribed customers
Segmented customers based on purchase history

💡 Conclusion

This project demonstrates an end-to-end data analytics workflow, showcasing skills in Python, SQL, and data visualization. It highlights the ability to transform raw data into actionable insights that can support business decisions.
